## Introduce

약품 설명서, QR 링크 등 일상에서 버리기도 아깝고, 그렇다고 다 모아 놓을 수도 없는 텍스트들이 굉장히 많습니다.
그래서 이런 텍스트를 사진으로 찍거나 QR을 스캔해서 해당 정보를 간편하게 저장할 수 있는 메모 앱을 만들어 보았습니다.
